# Steam Friend History

一個輕量級的 Raspberry Pi 專用系統，記錄自己的 Steam 好友列表與變化。

---

## 📦 功能
- 每天自動抓取 Steam 好友清單
- 自動備份每日快照
- 網頁界面顯示好友新增與刪除
- 好友暱稱變更記錄
- 國別統計與國別分類
- 點擊好友可以連到 Steam 個人頁面
- 使用 Docker 簡單部署

---
